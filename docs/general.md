
## SEARCH

 **n** – search for next occurrence (up)

 **N** – search for next occurrence down

 Need to escape special characters \

 / - search forward
 ? - search backward

 /\c – search case insensitive
 /\C – search case sensitive


## SUBSTITUDE:

> :4,7s/two/VIM
- begin with `:`
- from line **4** to **7**
- find string **two**
- replace with string **VIM**
